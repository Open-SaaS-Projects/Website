4. Applications Panel: Same Pattern

ApplicationsPanel will follow the same card-based pattern:

  • Desktop: Keep existing table with grid-cols-[1fr_180px_110px_100px_40px]
  • Mobile: Cards showing applicant name, job title, status badge, and tap-to-open drawer
  • Filters: Stack vertically on mobile instead of horizontal row
  • Drawer: Already works well on mobile (max-w-md, full-height)

5. Header Responsive Adjustments

  • Mobile: Reduce padding (px-4 py-3), smaller text (text-sm)
  • "Add position" button: Show icon only on mobile (< sm), full text on tablet+
  • Title/subtitle: Keep visible but reduce font size on mobile
